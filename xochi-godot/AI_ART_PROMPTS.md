## üöÄ Quick Start Workflow

### **Using DALL-E 3 (ChatGPT Plus)**

1. Copy prompt from above
2. Paste into ChatGPT
3. Add: "Generate 4 variations in different colors"
4. Download as PNG
5. Import to Godot `assets/sprites/` folder

### **Using Bing Image Creator (FREE)**

1. Go to: https://www.bing.com/create
2. Paste prompt
3. Click "Create"
4. Download all 4 variations
5. Choose best one

### **Using Stable Diffusion (FREE, Local)**

1. Install Automatic1111 WebUI
2. Use model: "pixel-art-xl" or "pixelart-diffusion"
3. Paste prompt
4. Add to negative prompt: "blurry, 3D, realistic, photograph"
5. Generate with settings:
   - Steps: 20-30
   - CFG Scale: 7
   - Sampler: Euler a
6. Upscale to exact pixel size needed

## üí° Tips for Best Results

### **For Pixel Art Sprites:**
- Always specify exact pixel dimensions (16x16, 32x32, etc.)
- Add "transparent background PNG" to every prompt
- Use "pixel art" + "2D game sprite" + "retro style"
- Specify "clean edges, no blur"
- Ask for "limited color palette"

### **For Sprite Sheets (Animations):**
- Specify "sprite sheet with X frames"
- Add "laid out horizontally" or "grid format"
- Mention "walk cycle" or "flight cycle"
- Can also generate single frames and animate in Godot

### **For Backgrounds:**
- Specify wide resolution (800px+ width)
- Add "parallax scrolling game background"
- Mention layer depth (distant/mid/foreground)
- Use "transparent sky" for easy layering

### **Color Consistency:**
- Generate all sprites in one session when possible
- Reference "Mexican folk art color palette"
- Save your favorite generations as style references
